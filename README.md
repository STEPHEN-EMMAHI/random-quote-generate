# Project Title and Description

A loveguide website that fetches and displays quotes about what to look for in a partner.
The ultimate goal of this project is to practice asynchronous API fetching. Currently using data(quotes) locally from an array of objects.

<!-- key features -->

## Key Features

1.  Random quote generation: Fetches data from the array of objects.
2.  Favorites System: You can save quotes to a personalised list.
3.  Show All Quotes: Allows you to display all quotes in the system.
4.  Clear All Quotes: Allows you to clear all of the quotes that has been displayed.
5.  Responsive Design: Fully functional on mobile, tablet and desktop.
6.  Category: Quotes are grouped into stages: Marriage and dating

## Installations

**No framework and package managers required.**

1. Clone the repository

```bash
git clone https://github.com/STEPHEN-EMMAHI/loveguide-random-quote-generator.git
```

OR

```bash
git clone git@github.com:STEPHEN-EMMAHI/loveguide-random-quote-generator.git
```

2. Navigate into the project folder

```bash
cd loveguide-random-quote-generator
```

3. Open the project :
   You can open index.html directly in your browser or use VS Code Live Server for better development experience.

## Requirements

- A modern web browser
- VS Code
- Live Server extension (optional)
