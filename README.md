<h1>Project Title & Description</h1>
A loveguide website that fetches and displays quotes about what to look for in a partner.
The ultimate goal of this project is to practice asynchronous API fetching and state management. Currently using data(quotes) locally from an array of objects.

<h2>Key Features</h2>
  <ul>
    <li>Random quote generation: Fetches data from the array of objects.
    </li>
    <li>Favorites System: You can save quotes to a personalised list.
    </li>
    <li>Show All Quotes: Allows you to display all quotes in the system.
    </li>
    <li>Clear All Quotes: Allows you to clear all of the quotes that has been displayed.
    </li>
  </ul>
